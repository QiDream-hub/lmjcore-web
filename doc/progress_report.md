## 一、项目概况

### 1.1 项目目标
基于 LMJCore 存储引擎开发一个轻量级 HTTP 网络服务壳，提供 RESTful API 接口，支持对象和集合的 CRUD 操作、链式查询、事务管理等功能。

### 1.2 当前状态
**第一阶段 (MVP) 已完成**，项目基础框架搭建完毕，核心功能已实现。
