🔧 LMJCore-Web API 测试工具

输入 LMJCore-Web 服务器的地址
💾 已保存的指针: 对象：- | 集合：-
1. 创建对象
POST/obj
2. 获取完整对象
GET/obj/{ptr}
3. 设置成员值
PUT/obj/{ptr}/{member}
支持：原始数据、指针引用 (01/02 开头 34 位十六进制)、null
4. 获取成员值
GET/obj/{ptr}/{member}
5. 删除成员
DELETE/obj/{ptr}/{member}
6. 链式查询
GET/obj/query?path={path}
格式：<指针>.<member1>.<member2>...
7. 创建集合
POST/set
8. 获取完整集合
GET/set/{ptr}
9. 添加元素
POST/set/{ptr}/elements
10. 删除元素
DELETE/set/{ptr}/elements
11. 检查指针是否存在
GET/ptr/{ptr}/exist
12. 健康检查
GET/health
